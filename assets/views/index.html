Visual Bookmark gives the user the ability to create, manage, and understand their bookmarks. This application is designed for any skill level. It is available for anyone who would like easy accessibility to their favorite web pages. This application is created for the user who would like to add or manage bookmarks for easy access and preview the page to get a live update of the web page. A search box is available to search through existing bookmarks. There is also a field where the user will be able to create their bookmarks.
