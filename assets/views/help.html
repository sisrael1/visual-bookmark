Visual Bookmark Help

Using Visual Bookmark

  • Click Visual Bookmark icon to add webpage to bookmark
  • Click wrench icon on bookmark card to edit bookmark information
  • Users are able to edit bookmark title and tags
  • Click ‘X’ icon to delete bookmark card

Tips for Visual Bookmark

  • Create your own titles for webpages
  • Create tags to find card quickly
  • Use colon to separate tags

Shortcuts for Visual Bookmark

  • Ctrl + Shift + V to add new bookmark card
  • Ctrl + Shift + D to delete card
  • Ctrl + Shift + E to edit card
  • Ctrl + Shift + S to open settings
